# res_pjsip_cisco_dnd
Asterisk PJSIP module for FreePBX 17 that enables the DND (Do Not Disturb) softkey on Cisco CP-8811, CP-8841, CP-8851, CP-8861 and other CP-8xxx IP phones. Intercepts proprietary pidf+xml PUBLISH requests and synchronizes the status with FreePBX's native AstDB DND state and BLF hints.
